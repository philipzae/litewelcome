Welcome to Linux Lite. Linux Lite is free for everyone to use and share, and suitable for people who are new to Linux or for people who want a lightweight environment that is also fully functional.

Linux Lite aims to provide each user with a basic collection of everyday tools, including a web browser, email client, media player, office suite, image editor and so on. To get the best possible start to using Linux Lite, browse through the information below.
